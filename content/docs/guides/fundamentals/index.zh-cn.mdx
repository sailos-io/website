欢迎使用 Sealos DevBox！在开始使用之前，让我们先了解一下这个强大平台的核心概念。

本章节将帮助你掌握 Sealos DevBox 的基本工作流程，从项目创建、开发、发布到最终部署，每一个环节我们都会详细介绍。通过这些基础知识的学习，你将能够充分利用平台提供的功能，提升开发效率。

## 核心组件

| 组件 | 描述 |
|-----------|-------------|
| 项目 | 作为应用程序的基础单元，项目将你的应用所需的所有内容（包括运行环境、依赖和配置）整合在一起，实现统一管理。 |
| 开发环境 | 基于云端的专业开发环境，支持通过 Cursor 等主流 IDE 直接访问，让你随时随地都能进行开发工作。 |
| 发布版本 | 将应用程序打包为标准的 OCI 镜像并进行版本管理，确保应用可以稳定部署和回滚。 |
| 部署上线 | 一键将应用程序部署到 Sealos Cloud，快速实现应用上线。 |

接下来，我们将通过以下几个章节，详细介绍如何使用这些核心组件，帮助你构建一个完整的开发工作流：

1. [新建项目](./fundamentals/create-a-project)：手把手教你在 Sealos DevBox 中创建和配置新项目。
2. [开发调试](./fundamentals/develop)：学习如何连接开发环境并开始编码工作。
3. [发布版本](./fundamentals/release)：掌握如何将应用打包为标准 OCI 镜像并进行版本管理。
4. [部署上线](./fundamentals/deploy)：了解如何将应用快速部署到云端并对外提供服务。

通过学习这些基础内容，你将能够熟练运用 Sealos DevBox 的各项功能，打造一个高效的开发工作流程。从最初的项目创建，到最终的应用部署，每一步都经过精心设计，让开发过程更加顺畅。

如果你想快速上手体验，也可以直接查看我们的[快速入门教程](/docs/quick-start)，通过实际操作来了解平台的核心功能！
