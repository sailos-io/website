MongoDB 是一个广受欢迎的开源文档型数据库系统。在 Sealos DevBox 中，您可以轻松地为您的开发项目设置和连接 MongoDB 数据库。

## 在 Sealos 中部署 MongoDB

Sealos 提供了简单直观的操作界面，只需几步即可完成 MongoDB 数据库的部署。具体步骤如下：

<div className='steps [&_h5]:step'>

<h5>打开 [Sealos 控制台](https://cloud.sealos.run)，找到并点击"数据库"应用图标。</h5>

![数据库服务入口](./images/sealos-postgresql-database-entry.zh-cn.jpg)

<h5>在数据库应用界面中，点击"新建"按钮，然后在部署表单中填写以下信息：</h5>

   - 数据库类型：选择 ”MongoDB”
   - 版本：选择所需的 MongoDB 版本 (如 mongodb-6.0)
   - 数据库名称：输入名称 (仅使用小写字母和数字)
   - 计算资源：通过滑块调整 CPU 和内存配置
   - 副本数量：开发测试环境建议设置为 1
   - 存储容量：设置所需的存储空间 (如：3 Gi)


![MongoDB 数据库配置界面](./images/sealos-mongodb-deployment-config.zh-cn.png)

<h5>确认配置无误后，您可以在左侧边栏查看预估成本，然后点击右上角的"部署"按钮开始创建数据库。</h5>

</div>

部署完成后，系统将提供数据库的连接参数，您可以使用这些信息连接到 MongoDB 数据库。

![MongoDB 数据库连接信息](./images/sealos-mongodb-connection-info.zh-cn.png)
